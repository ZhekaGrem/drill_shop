# Керівництво по колірній гамі проекту

## 🎨 Кольори

### Основні кольори

```css
#E0DDCA - Основний фон (беж)
#33603B - Темно-зелений (кнопки, футер, другорядний текст)
#E6DB1B - Жовтий (ховер кнопок, кнопки 3-го рівня)
#2B2B27 - Темний текст на світлому фоні
#A63C48 - Червоний (ховер посилань, ціни зі знижкою, помилки)
```

### Правила використання

1. **Фон `#E0DDCA`** → Текст `#2B2B27`
2. **Фон `#33603B`** → Текст `#E0DDCA`
3. **Ховер на посилання** → `#A63C48`
4. **Ціни зі знижкою** → `#A63C48`

## 🔘 Варіанти кнопок

### 1. Головні кнопки (Primary) - `variant="filled"`

**Використання:** Основні дії (Купити, Оформити замовлення, Додати в кошик)

```tsx
<Button variant="filled">Купити зараз</Button>
```

**Стилі:**
- Background: `#33603B`
- Text: `#E0DDCA`
- **Hover:** background `#E6DB1B` + text `#2B2B27`

### 2. Другорядні кнопки (Secondary) - `variant="light"`

**Використання:** Другорядні дії (Переглянути, Детальніше, Скасувати)

```tsx
<Button variant="light">Переглянути</Button>
```

**Стилі:**
- Background: `#E0DDCA`
- Text: `#33603B`
- **Hover:** background `#E6DB1B` + text `#2B2B27`

### 3. Кнопки третього рівня (Tertiary) - `variant="outline"`

**Використання:** Менш важливі дії

```tsx
<Button variant="outline">Фільтри</Button>
```

**Стилі:**
- Background: `#E6DB1B`
- Text: `#2B2B27`
- **Hover:** text `#33603B` (фон не міняється)

## 📝 Типографія

### H1 заголовки

```tsx
<h1>Заголовок</h1>
```

**Стилі:**
- Font: Rubik
- Weight: 900 (Black)
- Size: 40px
- Line height: 120%
- Color: `#2B2B27` на світлому / `#E0DDCA` на темному

### Текст кнопок

```css
Font: IBM Plex Sans Condensed
Weight: 700 (Bold)
Size: 20px
Line height: 120%
Letter spacing: 4%
Case: UPPERCASE
```

## 🎯 CSS Змінні

Використовуйте в коді:

```css
/* Фон */
--background: #e0ddca
--background-secondary: #33603b

/* Основні кольори */
--primary: #33603b
--secondary: #e0ddca
--accent: #e6db1b
--accent-red: #a63c48

/* Текст */
--text-primary: #2b2b27 (на світлому)
--text-inverse: #e0ddca (на темному)
--text-secondary: #33603b
```

## 🛠️ Utility класи

```css
.text-on-light     /* #2B2B27 - Темний текст на світлому */
.text-on-dark      /* #E0DDCA - Світлий текст на темному */
.text-secondary    /* #33603B - Другорядний текст */
.text-accent-red   /* #A63C48 - Червоний для ховерів, помилок, цін зі знижкою */
.btn-text          /* Стилі тексту кнопок */
```

## 📋 Приклади використання

### Кнопки в компонентах

```tsx
import { Button } from '@mantine/core';

// Головна кнопка
<Button variant="filled">Додати в кошик</Button>

// Другорядна кнопка
<Button variant="light">Детальніше</Button>

// Кнопка 3-го рівня
<Button variant="outline">Показати фільтри</Button>
```

### Текст на різних фонах

```tsx
// На світлому фоні (#E0DDCA)
<Text className="text-on-light">Темний текст</Text>

// На темному фоні (#33603B)
<Text className="text-on-dark">Світлий текст</Text>

// Другорядний текст
<Text className="text-secondary">Зелений текст</Text>

// Червоний текст (ціна зі знижкою)
<Text className="text-accent-red">-20%</Text>
```

### Посилання

```tsx
// Звичайне посилання - автоматично має червоний ховер
<a href="/catalog">Каталог</a>
```

## 🎨 Footer

Footer автоматично використовує темний фон `#33603B` зі світлим текстом `#E0DDCA`.

Всі посилання у футері мають червоний ховер `#A63C48`.

---

## ❌ Помилки

Всі помилки в інпутах та нотифікаціях автоматично використовують колір `#A63C48`.

### Приклади:

```tsx
// Інпут з помилкою
<TextInput
  label="Email"
  error="Невірний email"
/>

// Нотифікація помилки
<Notification color="red" title="Помилка">
  Щось пішло не так
</Notification>
```

## ⚠️ Важливо

1. **НЕ** використовуйте інші кольори без узгодження
2. **ЗАВЖДИ** перевіряйте контрастність тексту на фоні
3. **ВИКОРИСТОВУЙТЕ** варіанти кнопок згідно з їх призначенням
4. **НЕ** змінюйте типографію кнопок
